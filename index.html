Introduction

The terminal for all developers is a tool very important, because in this tool we can update packages, install more tools, navigate into directories, deploy our applications, programming with terminal editors and more...

In this documentation I recopile some information that maybe can help you to understand how you can use your terminal bash every day

Ubuntu

Ubuntu is a GNU/Linux distribution Debian based, is an operating system open source and free, in this documentation I will show you how to use an Gnome Terminal

Gnome is an type of desktop, exist another types of desktops that you can use with your favorite distribution, like XFCE, KDE among others

Manager Packages

Let's imagine you want to update your programs or you system for more security, for this the package manager can help you

Exist many package manager like brew in MacOs, pacman for Arch linux distributions, yarn for Fedora but Ubuntu and Debian based distributins uses the package manager "apt"

If you need update completly your system I recommended you write the next command in your terminal:

sudo apt-get update -y && sudo apt-get upgrade -y

If you want clean your packages and your innesesary packages you can write:

sudo apt-get autoclean && sudo apt-get autoremove -y
Navigation

Basiclly you can navigate with a few commands, for example:

You can list the information into a directory with the command:

ls

If you want enter into a directory you can write the next command:

cd name_directory

If you want to leave a directory you can write the next command:

cd ..

In the case that you want create a directory:

mkdir name_directory

But if you want remove a directory

rm -r name_directory

The next command is for create files

touch name_file

For remove files

rm name_file
Editors

In linux exist many terminal editors, but the most popular are nano and vim

Personally I like vim because is many fast, minimlist and simple, you really can create a programm or a web application with this editor with help of plugins

You can install neovim this is exactly vim but current

sudo apt-get install neovim -y

for edit a file for example:

nvim name_file

into neovim you can navigate up with the keyboard

k

down with:

j

right with:

l

left with

h

for write into a file with vim you can press

i

this mode is for insert, if you want navigate into a file you should press the key

esc

if you want save a file

:w

for exit of vim

:q
